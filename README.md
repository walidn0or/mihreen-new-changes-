# Mihreen LLC Website

This repository contains the Next.js website for Mihreen LLC, built with React, Tailwind CSS, and Next.js 16.

## What’s included

- SEO-ready metadata in `app/layout.tsx`
- `robots.txt` and `sitemap.xml` generated automatically by Next.js
- `app/services/page.tsx` and `app/about/page.tsx` routes for public pages
- Static export-ready homepage and sections

## Deployment

1. Install dependencies:
   ```bash
   npm install
   ```
2. Build the project:
   ```bash
   npm run build
   ```
3. Start in production mode:
   ```bash
   npm run start
   ```

## Repository setup

The package metadata is configured for the repository:

- `name`: `mihreen-final`
- `homepage`: `https://www.mihreenllc.us`
- `repository`: `https://github.com/walidn0or/mihreen-final.git`
